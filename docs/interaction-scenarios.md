# Interaction Scenarios

## Shared Scenarios

### Select by Pointing and Speech

1. User points at an object.
2. User says "select this".
3. The system resolves the pointed object.
4. The object becomes selected.

### Move by Grab and Placement

1. User grabs or pinches an object.
2. User says "move this here" or drags the object.
3. The system resolves the source object and target position.
4. The object is moved.

### Rotate Selected Object

1. User selects an object.
2. User says "rotate left" or performs a rotation gesture.
3. The system applies a rotation action.

### Delete with Confirmation

1. User points to or selects an object.
2. User says "delete this".
3. The system confirms if confidence is low or deletion is destructive.
4. The object is removed.

## Shape Puzzle Scenarios

- place a shape into a matching slot
- rotate a shape until it matches orientation
- confirm placement by speech

## Virtual Workspace Scenarios

- create an object
- move objects around a free-form board
- arrange multiple objects spatially
- delete or re-select objects during editing
